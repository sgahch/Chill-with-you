# Chill With You: Lo-Fi Story

> 与你的悠闲时光：Lo-Fi故事

## 📖 简介

《Chill With You: Lo-Fi Story》是一款由 **Nestopi (Bulbul工作室)** 开发的休闲效率类游戏。游戏融合了番茄钟工作法与 Lo-Fi 音乐氛围，帮助玩家在轻松的环境中专注工作和学习。

## 🎮 游戏特色

- **🍅 番茄钟计时器**：内置番茄工作法计时功能，帮助管理工作与休息时间
- **📝 任务管理**：可创建和跟踪待办任务
- **🎵 Lo-Fi 音乐**：沉浸式的 Lo-Fi 背景音乐，营造专注氛围
- **🎧 自定义音乐**：支持导入自己喜欢的音乐
- **🌅 场景切换**：多种环境背景可供选择
- **🐱 可爱角色**：陪伴你一起工作的虚拟角色
- **📚 小说阅读**：内置故事内容

## 💻 系统要求

- **操作系统**：Windows
- **运行环境**：需要安装运行库支持
- **引擎**：Unity

## 🚀 如何启动

1. 双击 `启动游戏.exe` 启动游戏
2. 或直接运行 `Chill With You.exe`
